# Namma Madras — Landing

Marketing site for **Namma Madras** (நம்ம மெட்ராஸ்): React, Vite, Tailwind CSS v4, Framer Motion.

## Scripts

```bash
npm install
npm run dev    # http://localhost:5173
npm run build
npm run preview
```

## Deploy (Vercel)

Import the repo in [Vercel](https://vercel.com): framework **Vite**, build `npm run build`, output `dist`. No extra config required.
